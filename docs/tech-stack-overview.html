Core Infrastructure

The foundation powering the storefront, real-time data, and content management.

Frontend Framework

React-based framework with App Router, server-side rendering, and static generation. Powers every page from the catalog to the B2B portal with blazing-fast navigation.
App Router SSR + SSG API Routes React 19 TypeScript

Hosting & Deployment
Vercel
Global edge network with automatic CI/CD. Every git push deploys instantly with preview URLs for review before going live.
Edge Network Auto Deploy Preview URLs
C
Real-Time Database & Backend
Convex
Serverless database with real-time subscriptions. Powers the product catalog, Grace AI conversations, portal orders, and form submissions — all updating live.
Real-Time Sync Serverless Functions Full-Text Search
S
Content Management
Sanity.io
Headless CMS for editorial content. The marketing team manages the mega-menu, homepage hero, blog journal, and product family cards — no developer needed.
Sanity Studio Mega Menu Blog / Journal
S
E-Commerce Engine
Shopify
Handles checkout, payment processing, inventory, and order fulfillment. The storefront resolves SKUs to Shopify variants via the Admin API for seamless cart-to-checkout.
Admin API Checkout Inventory Webhooks

02 / AI & Intelligence

AI & Intelligence Layer

Grace, the AI sales assistant, combines conversational AI with deep product knowledge to guide buyers.

G
AI Sales Assistant
Grace AI
A conversational assistant trained on the full product catalog, fitment data, packaging trends, and B2B sales strategies. Grace can search 2,354 SKUs, check component compatibility, recommend products by use case, and guide buyers through the ordering process — via text or voice.
Product Search Fitment Matching Voice Conversations B2B Personas Trend Intelligence
A
Language Model
Claude (Anthropic)
Claude Sonnet 4 powers Grace's reasoning. It processes product queries, understands packaging context, and generates nuanced recommendations with tool-use capabilities.
Sonnet 4 Tool Use Structured Output
11
Voice AI
ElevenLabs
Provides natural voice conversations for Grace. Buyers can speak to Grace instead of typing — ask about products, hear recommendations, and get guided through orders hands-free.
Conversational AI Voice Streaming Text Fallback
C
Authentication & Identity
Clerk
Manages B2B portal authentication with organization-level accounts. Wholesale buyers sign in to view orders, manage drafts, access Grace workspace, and track shipments.
SSO / OAuth Organizations Portal Access

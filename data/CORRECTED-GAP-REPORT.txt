PARSING BUGS FOUND IN ORIGINAL CRAWL
--------------------------------------------------------------------------------

BUG 1: CAPACITY PARSING
  Legacy SKU codes embed capacity as a number suffix before the finish code.
  The crawler was extracting only partial digits:

  GBCrcl15BlkShSht  → was parsed as 5ml,  correct: 15ml
  GBCrcl50RdcrBlkLthr → was parsed as 5ml, correct: 50ml
  GBCrcl100RdcrBlkLthr → was parsed as 10ml, correct: 100ml
  GBDiva46RdcrBlkLthr → was parsed as 6ml, correct: 46ml
  GBElg60RdcrBlkLthr → was parsed as 6ml, correct: 60ml
  GBRnd78RdcrBlkLthr → was parsed as 8ml, correct: 78ml
  GBRnd128RdcrBlkLthr → was parsed as 8ml, correct: 128ml
  GBGrce55RdcrBlkLthr → was parsed as 5ml, correct: 55ml
  (and 33 more with same pattern)

  Impact: 41 of 61 "gaps" were false positives from wrong capacity matching

BUG 2: APPLICATOR PARSING
  The suffix "RdcrBlkLthr" means "Reducer + Black Leather cap" — a reducer product.
  The crawler was classifying these as "cap-only" when they're actually reducers.

  GBDiva30RdcrBlkLthr = Diva 30ml Clear REDUCER  → matches diva-30ml-clear-18-415-reducer ✅
  GBSlk30RdcrBlkLthr  = Sleek 30ml Clear REDUCER → matches sleek-30ml-clear-18-415-reducer ✅
  GBSlm30RdcrBlkLthr  = Slim 30ml Clear REDUCER  → matches slim-30ml-clear-18-415-reducer ✅

  Legacy SKU suffix decoder:
    BlkShSht    = Black Short cap (cap-only)
    RdcrBlkLthr = Reducer + Black Leather cap (reducer applicator)
    SpryBlkMatt = Sprayer Black Matte (fine mist sprayer)
    RollBlkDot  = Roll-on Black Dot (roll-on)
    GlCap       = Glass Cap (cap-only)

FAMILY NAME MAPPING
--------------------------------------------------------------------------------
The live site uses "Tall Cylinder" and "Tall Rectangle" as family names.
Convex stores these differently:

  Live Site             Convex Family     Convex Slug Pattern
  ───────────────────   ──────────────    ──────────────────────────────
  Tall Cylinder         Cylinder          cylinder-9ml-*-13-415-*
  Tall Rectangle        Rectangle         tall-rectangle-10ml-clear-13-415-*
  Pillar                Pillar            pillar-9ml-clear-*

  Note: "Tall Cylinder" 9ml with 13-415 threading = same as Cylinder 9ml small-thread
  Convex uses the base family name "Cylinder" but differentiates by thread size in slug.

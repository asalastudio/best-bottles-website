# Paper Doll — CYL-9ML pilot

- **`CYL-9ML/manifest.json`** — generated by `npm run paper-doll:manifest` with `PAPER_DOLL_ASSETS_ROOT` pointing at the folder that contains `family-model.json` and `bodies/`, `caps/`, etc. Committed so the render API can fall back when Sanity is unavailable.
- **Sanity** — run `npm run paper-doll:upload` with the same env plus `SANITY_API_TOKEN` to upload PNGs and create/update the `paperDollFamily` document.
- **Render API** — `POST /api/paper-doll/render` with JSON body, e.g. roll-on:

```json
{
  "family": "CYL-9ML",
  "mode": "rollon",
  "body": "CLR",
  "roller": "MTL-ROLL",
  "cap": "SHN-BLK"
}
```

Set `"preview": true` for a max-width 800px PNG.
